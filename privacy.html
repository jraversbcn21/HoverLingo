Política de Privacidad de HoverLingo

Fecha de entrada en vigor: 23 de julio de 2026

HoverLingo es una extensión de Chrome que traduce palabras y frases al colocar el cursor sobre ellas. Esta política describe qué datos procesa la extensión y cómo.

Datos que se procesan

  • Texto de la página: cuando colocas el cursor sobre una palabra (o seleccionas un fragmento), esa palabra y la frase que la rodea se envían a la API de Groq (api.groq.com) con el único fin de generar la traducción. Este envío ocurre solo cuando la extensión está activada y solo para el texto sobre el que interactúas — nunca se envía la página completa.
  • Clave de API: tu clave de API de Groq, que tú mismo proporcionas, se guarda únicamente en el almacenamiento local de tu navegador (chrome.storage.local). Solo se transmite a api.groq.com como autenticación de tus propias peticiones. Nunca se exporta ni se comparte.
  • Ajustes y estadísticas de uso: tus preferencias (idioma, modelo, sitios desactivados, etc.) y contadores locales de uso (número de palabras traducidas, tasa de aciertos de caché) se guardan únicamente en tu dispositivo.
  • Caché de traducciones: las traducciones recientes se guardan localmente (con una caducidad de 24 horas) para evitar peticiones repetidas.

Lo que NO hacemos

  • No recopilamos, almacenamos ni vemos ningún dato tuyo: la extensión no tiene servidor propio.
  • No usamos analítica, publicidad ni rastreadores de ningún tipo.
  • No vendemos ni cedemos datos a terceros.
  • No sincronizamos tu clave de API con la nube (se guarda en storage.local, no en storage.sync).

Terceros

Las traducciones las genera Groq, Inc. El texto enviado se procesa según la política de privacidad de Groq. HoverLingo no controla ese procesamiento; si no estás de acuerdo, no configures una clave de API.

Permisos de la extensión

  • Acceso a las páginas web: necesario para detectar la palabra bajo el cursor y mostrar el tooltip de traducción en cualquier página. La extensión no lee ni transmite el contenido de la página más allá del texto sobre el que pasas el cursor.
  • storage: para guardar tus ajustes, la caché y las estadísticas localmente.
  • api.groq.com: el único servidor con el que se comunica la extensión.

Control y eliminación de datos

Puedes desactivar HoverLingo globalmente o por sitio desde el popup. Desinstalar la extensión elimina todos los datos locales (clave de API, ajustes, caché y estadísticas).

Cambios y contacto

Si esta política cambia, la nueva versión se publicará en esta misma URL. Para cualquier consulta: sidmaierlabs@gmail.com.

HoverLingo Privacy Policy

Effective date: July 23, 2026

HoverLingo is a Chrome extension that translates words and phrases when you hover over them. This policy describes what data the extension processes and how.

Data processed

  • Page text: when you hover over a word (or select a snippet), that word and its surrounding sentence are sent to the Groq API (api.groq.com) for the sole purpose of generating the translation. This happens only while the extension is enabled and only for the text you interact with — the full page is never sent.
  • API key: your self-provided Groq API key is stored only in your browser's local storage (chrome.storage.local). It is transmitted only to api.groq.com to authenticate your own requests. It is never exported or shared.
  • Settings and usage stats: your preferences (language, model, disabled sites, etc.) and local usage counters (words translated, cache hit rate) are stored only on your device.
  • Translation cache: recent translations are cached locally (24-hour expiry) to avoid repeated requests.

What we do NOT do

  • We do not collect, store, or see any of your data: the extension has no server of its own.
  • No analytics, no advertising, no trackers of any kind.
  • We do not sell or share data with third parties.
  • Your API key is never synced to the cloud (it lives in storage.local, not storage.sync).

Third parties

Translations are generated by Groq, Inc. Submitted text is processed under Groq's privacy policy. HoverLingo does not control that processing; if you disagree with it, do not configure an API key.

Extension permissions

  • Access to web pages: required to detect the word under your cursor and render the translation tooltip on any page. The extension does not read or transmit page content beyond the text you hover over.
  • storage: to keep your settings, cache, and stats locally.
  • api.groq.com: the only server the extension communicates with.

Data control and deletion

You can disable HoverLingo globally or per site from the popup. Uninstalling the extension removes all local data (API key, settings, cache, and stats).

Changes and contact
